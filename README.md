# Food Store - Sistema de Gestión de Pedidos de Comida

**Materia:** Programación 2

## Requisitos y Ejecución
- **Lenguaje:** Java 21
- **IDE recomendado:** NetBeans (con Maven)
- **Ejecución:** Clonar el repositorio y ejecutar la clase `Main.java` ubicada en el paquete `integrado.prog2`. Toda la interacción se realiza mediante la consola del IDE.
